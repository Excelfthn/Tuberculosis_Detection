# Git Setup and Push Commands

## Step 1: Navigate to your project directory
cd "C:\claude-code\PCD_Final_Project"

## Step 2: Initialize Git repository (if not already done)
git init

## Step 3: Add your GitHub repository as remote origin
git remote add origin https://github.com/Excelfthn/Tuberculosis_Detection.git

## Step 4: Add all files to staging area
git add .

## Step 5: Commit your changes
git commit -m "Initial commit: TB Detection project with 92.86% accuracy

Features:
- Complete image preprocessing pipeline with CLAHE
- Lung segmentation using Otsu thresholding
- GLCM + LBP feature extraction
- SVM classifier with RBF kernel
- Interactive visualization with TB probability overlay
- Comprehensive README and documentation"

## Step 6: Push to GitHub
git branch -M main
git push -u origin main

## Alternative: If repository already exists on GitHub and has content
git pull origin main --rebase
git push origin main
